Welcome to Namaste Angular App

This is a simple Angular application showcasing various features and functionalities.

Feel free to explore the different sections of the app and try out its features.

Features

  • Authentication: Login and register functionality.
  • CRUD Operations: Perform CRUD operations on a list of items.
  • Forms: Example forms with validation.
  • Routing: Navigate between different pages.
  • HTTP Requests: Make HTTP requests to a mock server.
